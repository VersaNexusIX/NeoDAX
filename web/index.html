// Core Features
Disassembler
x86-64, AArch64, RISC-V RV64GC. Symbol resolution, C++ demangling, instruction group coloring.
-y -d -g -a
CFG Builder
Two-pass algorithm with jump-trick awareness. Loop detection via dominator analysis.
-C -L
Decompiler & SSA
ARM64 SSA lifting with versioned register assignments. Pseudo-C output from IR.
-Q -D
ARM64 Emulator
Concrete execution with 32 GP registers, 64 memory pages, CPSR flags. Step-by-step trace.
-I
Entropy Analysis
Shannon entropy sliding window. Detects packed, encrypted, compressed sections automatically.
-e
JavaScript API
Native N-API addon. TypeScript declarations included. Web UI with 16 analysis panels.
